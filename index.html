Informações da Camada

Selecione uma camada no menu à esquerda para visualizá-la no mapa.

Instruções:

  • Use o menu de camadas para ativar/desativar camadas
  • Clique em qualquer feature vetorial para ver detalhes
  • Use o botão "Resetar Vista" para voltar à visão inicial
  • Alternar para vista de satélite com o botão no canto inferior direito
  • Grupos de camadas:
    • Vetoriais: Bacia, Sub-bacias e Hidrografia
    • Históricos: Mapas de 1967 e 1976
    • Batimetria: Levantamentos de 1995 a 2024
    • Uso do Solo: Classificação LULC
  • Nota: As camadas raster (históricos, batimetria e LULC) são mutuamente exclusivas para melhor visualização.
